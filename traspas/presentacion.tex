\begin{hslide}
\slsect{Objetivo}
Crear versiones web (multiplataforma) de varios clientes 
de JdeRobot muy utilizados sin tener servidor intermedio. 

Sub-objetivos:
\begin{itemize}
\item Representación de datos de sensores: CameraView, RgbdViewer
\item Interactuación con actuadores: KobukiViewer, UavViewer
\item Añadir comportamiento autónomo
\item Diseño de la página web
\end{itemize}

\end{hslide}

\begin{hslide}
\slsubsect{IntrorobKobukiJS}
\begin{itemize}
\item Igual que KobukiViewerJS pero añadiendo la posibilidad de añadir comportamiento autónomo programado en JavaScript. 
\item Cada 10ms (usando \texttt{setInterval}) se envían los datos de los sensores a un worker y éste devuelve las órdenes para los actuadores.
\end{itemize}

\begin{hslide}
\slsect{Objetivo}
Crear versiones web de varios clientes 
de JdeRobot muy utilizados sin tener servidor intermedio, permitiéndo tener versiones multiplataforma con un único desarrollo. 

Sub-objetivos:
\begin{itemize}
\item Representación de datos de sensores: CameraView, RgbdViewer
\item Interactuación con actuadores: KobukiViewer, UavViewer
\item Añadir lógica que interprete los datos de los sensores y mande órdenes a los actuadores
\item Diseño de la página web
\end{itemize}

\end{hslide}
